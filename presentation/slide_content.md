## Slide 2: Редактор как в Notion
**Удобный редактор с автоформатированием**

Полнофункциональный WYSIWYG редактор с поддержкой:
- Форматирование текста: жирный, курсив, подчёркивание, зачёркивание
- Заголовки H1-H6, списки, блоки кода
- Таблицы с редактированием ячеек
- Изображения и видео (S3, YouTube, RuTube, VK)
- Цитаты, ссылки, чек-листы

Автоформатирование при вводе:
- `#` → Заголовок
- `-` или `*` → Маркированный список
- `1.` → Нумерованный список
- `[]` → Чек-лист
- ``` → Блок кода

## Slide 4: Интеллектуальный поиск
**Два режима поиска: текстовый и AI**

Текстовый поиск:
- Точное совпадение по ключевым словам
- Поиск в заголовках и содержимом
- Мгновенные результаты
- Подсветка найденных фрагментов

AI-поиск (семантический):
- Понимание смысла запроса
- Поиск по контексту, а не только по словам
- Векторные embeddings через Ollama
- Находит релевантный контент даже без точных совпадений

Пример: запрос «как настроить доступ» найдёт статьи о правах, группах и ролях

## Slide 5: AI-помощник для написания
**9 инструментов для создания контента**

Функции AI-помощника:
1. **Improve Writing** — Улучшение стиля и читаемости
2. **Expand Text** — Расширение кратких заметок в полные статьи
3. **Summarize** — Создание кратких резюме
4. **Fix Grammar** — Исправление грамматики и орфографии
5. **Change Tone** — Изменение тона (формальный/неформальный/технический)
6. **Extract Keywords** — Извлечение ключевых слов для тегов
7. **SEO Optimize** — Оптимизация для поиска
8. **Create Outline** — Генерация структуры статьи
9. **Generate Content** — Создание контента из плана

Все AI-функции работают локально через Ollama — данные не покидают сервер!

## Slide 6: Управление доступом
**Гибкая система прав и ролей**

Роли пользователей:
- **Admin** — Полный доступ к системе и настройкам
- **Editor** — Создание и редактирование страниц
- **Reader** — Только чтение страниц
- **Guest** — Доступ к публичным страницам без авторизации

Уровни доступа к страницам:
- **Public** — Видна всем, включая гостей
- **Group** — Видна только членам группы
- **Private** — Видна только владельцу

Интеграция с Authentik:
- OAuth2 авторизация
- Синхронизация групп из Authentik
- Единый вход (SSO)

## Slide 10: Развёртывание и архитектура
**Готовое решение для self-hosted установки**

Технологический стек:
- Frontend: React, TypeScript, TailwindCSS
- Backend: Node.js, Express, tRPC
- База данных: PostgreSQL
- Хранилище: MinIO (S3-совместимое)
- AI: Ollama (локальные модели)
- Авторизация: Authentik OAuth2
- Reverse Proxy: Traefik с SSL

Docker Compose включает:
- Все необходимые сервисы
- GPU поддержка для AI
- Автоматический HTTPS
- Health checks и автоперезапуск

Развёртывание за 5 минут:
1. Клонировать репозиторий
2. Настроить config.env
3. Запустить docker-compose up
4. Загрузить AI модели
